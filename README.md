# django-api-todo

### This is a simple todo app made with django rest framework.

Every todo item contains creation time, title, weather it is completed or not and owner of the task. Session Authentication is used in this project.
basic CRUD operation is allowed.

### Valid Urls

admin/
api/tasks [name='list']
api/update/<int:id>/ [name='update']
api/delete/<int:id>/ [name='delete']
auth/
